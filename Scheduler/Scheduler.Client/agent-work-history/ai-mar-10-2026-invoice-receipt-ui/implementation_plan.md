# Invoice & Receipt Generation ‚Äî Implementation Plan (v4)

## Background

The PHMC Rec Committee needs **rental invoices** and **payment receipts**. The existing system tracks event charges and financial transactions but has no invoice/receipt workflow.

> [!NOTE]
> **Correction from v3:** All existing financial entities **are** already defined in `SchedulerDatabaseGenerator.cs` (verified via PowerShell scan of all 95 `AddTable` calls). Only net-new entities need to be added.

---
