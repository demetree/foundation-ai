PostgreSQL is an open-source relational database management system with a heritage going back to the Ingres research project at Berkeley in the 1980s. Distinguished from many of its peers by uncompromising adherence to the ACID transactional properties and to standard SQL, it has historically been the database of choice for workloads where correctness, durability, and complex queries matter more than raw throughput. The system supports an unusually rich type system including arrays, ranges, JSON, geometric primitives, and user-defined composite types, with first-class extensibility through a plugin mechanism that has produced industrial-strength extensions for full-text search, geospatial indexing (PostGIS), time-series workloads (TimescaleDB), and vector similarity search (pgvector). Foreign data wrappers expose external sources -- other databases, REST APIs, file formats -- as ordinary tables. The project is governed by an independent global development group rather than a single corporate sponsor, which has insulated it from the licensing controversies that have affected several other widely deployed databases.
