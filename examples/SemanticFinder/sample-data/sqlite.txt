SQLite is the most widely deployed database engine in the world, embedded in essentially every smartphone, desktop operating system, web browser, aircraft, and significant fraction of all running software in some form. Implemented as a single C source file of around 150,000 lines that compiles to roughly half a megabyte of object code, it provides a complete relational engine with full SQL support, ACID transactions, and a database-on-disk format that is itself a stable, public, cross-platform interchange format approved by the US Library of Congress for long-term preservation. Unlike a server-based database it has no separate process: the engine runs inside the application that uses it and operates directly on the database file, which makes it perfect for application-local persistence but a poor fit for write-heavy multi-user workloads. The project is in the public domain rather than open source in the licence sense, and the core team accepts no outside code contributions in order to keep ownership of the airworthiness-grade testing suite that distinguishes the project.
