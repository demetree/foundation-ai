Kubernetes is an open-source platform for automating the deployment, scaling, and operation of containerised applications across a cluster of machines. Originating at Google in 2014 as a re-architecture of the internal Borg system, it was donated to the Cloud Native Computing Foundation and has since become the dominant orchestration substrate for modern distributed software. The system is built around a declarative model in which the user describes the desired state of the workload -- replica counts, network exposure, persistent storage, scheduling constraints -- and a control loop continuously reconciles reality toward that description. Core building blocks include Pods (one or more co-located containers sharing a network namespace), Deployments (managed groups of replica Pods), Services (stable network identities for sets of Pods), and Ingresses (HTTP routing). Every major public cloud now offers a managed control plane, and a thriving ecosystem of operators extends the platform to manage databases, message queues, machine-learning pipelines, and almost every other category of stateful workload.
