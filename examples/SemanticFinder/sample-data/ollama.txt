Ollama is an open-source tool that packages popular open-weight language models behind a simple HTTP server, making it trivial to run them on your own desktop or laptop without sending any data to a remote service. After installing the binary you fetch a model with a single command, and then any application that speaks the OpenAI chat completions protocol can talk to localhost as if it were the cloud API. The project supports CPU inference everywhere and accelerates with CUDA on NVIDIA cards, Metal on Apple silicon, and ROCm on supported AMD hardware. Models live in a content-addressed local cache so multiple front-ends share one copy on disk. Memory pressure is managed by an automatic load-and-evict cycle: a model is brought into RAM on the first request, kept warm for a few minutes of idle, then released. The project's appeal is the friction-free path from "install" to "running a 7-billion-parameter model offline" in under five minutes.
