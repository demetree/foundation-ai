Docker is a software platform that bundles applications and their dependencies into self-contained, portable images that run identically on any machine with the runtime installed. Released in 2013, it dramatically simplified the previously specialist art of operating-system-level virtualisation by hiding the underlying Linux primitives -- namespaces, cgroups, union filesystems -- behind a friendly command-line interface and a declarative build file called the Dockerfile. The image format the project introduced has since been standardised by the Open Container Initiative and is now used by essentially every container tool and platform, even those built explicitly to compete with Docker. The Docker Engine itself bundles a build subsystem, an image store, a network and storage layer, and a runtime that ultimately delegates the actual process isolation to a lower-level component called containerd. Docker Desktop wraps all of this into a polished GUI application for Mac and Windows developers, and remains the default on-ramp to the broader cloud-native ecosystem.
