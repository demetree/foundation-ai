# SchedulerDatabase — Overview

SchedulerDatabase is a **.NET 10** class library containing the Entity Framework Core model for the Scheduler application's database.

---

## Contents

| Folder | Files | Purpose |
|--------|-------|---------|
| `Database/` | 140 | EF Core entity classes and `SchedulerContext` DbContext |
| `EntityExtensions/` | 138 | Partial class extensions for each entity |
| `Utility/` | 2 | Database utility helpers |

> [!CAUTION]
> The `Database/` folder is **auto-generated** by EF Core Power Tools reverse-engineering. Do not manually edit files in this folder — they will be overwritten when the model is regenerated.

---

## Key Files

| File | Purpose |
|------|---------|
| `Database/SchedulerContext.cs` | DbContext with DbSet properties for all 140 entities |
| `efpt.config.json` | EF Core Power Tools configuration for reverse-engineering |

---

## EntityExtensions

The `EntityExtensions/` folder contains **partial class files** — one per entity — generated by `SchedulerTools`. These add Foundation-specific functionality (change history, audit integration) to the EF entity classes without modifying the auto-generated database files.

---

## How It's Generated

1. Database tables are defined in `SchedulerDatabaseGenerator/`
2. SQL scripts are generated and applied to SQL Server
3. EF Core Power Tools reverse-engineers the database into the `Database/` folder
4. `SchedulerTools` generates the `EntityExtensions/` partial classes
