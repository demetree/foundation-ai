# AlertDatabase

This is a system built on .NET 9 that provides data structures to Quarterback system projects built on the .NET 9 platform.

Its purpose is to serve as a referencable library for higher level Quarterback projects.

The code it exposes that can be used by projects referencing this project includes:
	- The Entity Framework database context for the Quarterback schema.
	- A set of entity extensions that extend the Quarterback database context with more anonymous objects so they can be used for data transfer scenarios such as Web APIs.
