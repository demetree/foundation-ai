# HangFireDatabaseGenerator

This is a simple single source file project that defines the 3rd party HangFire system's database.  It is of limited use outside of the 
custom SQLite based implementation of HangFire in the FoundationFramework project.  This is an edge case that is unlikely to be used in 
operational systems.

This is a *.Net Standard class library that uses *CodeGenerationCommon*'s database generator to implement the 3rd party HangFire system's tables.
