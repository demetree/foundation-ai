# CodeGenerationCommon

This is a .Net Standard class library that contains a few classes that are cross version (.Net Framework and .Net Core) that provide the lowest
level base classes for Code Generation class implementations, as well as whatever else can be done in a platform independent capacity for 
code generation purposes.

It is the home to the main DatabaseGenerator class that is used extensively in Foundation applications for system defininition purposes.

It is to be referenced by higher level Code Generation libraries.
